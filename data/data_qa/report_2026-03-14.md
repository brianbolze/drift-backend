## New Warnings

### W8 — Berger ammunition product pages stored as bullet records
**12 records** | First seen: 2026-03-14

Berger loaded ammunition product pages (e.g., "300 Winchester Magnum 185 Grain Classic Hunter Rifle Ammunition") were extracted as bullet records. These duplicate the component bullet data (same BC, weight, diameter) but pollute the bullet table. Not causing incorrect ballistic calculations since specs match, but inflates bullet count and creates duplicate search results.

Affected records:
- 223 Rem 73gr BT Target Ammo
- 300 Norma 215gr Hybrid OTM Tactical Ammo
- 300 Norma 230gr Hybrid OTM Tactical Ammo
- 300 WM 168gr Classic Hunter Ammo

### W9 — Hornady 6.5CM 100gr ELD-VT source URL mismatch
**1 record** | First seen: 2026-03-14

Cartridge "6.5 Creedmoor 100 gr ELD-VT V-Match" has source_url with slug `6mm-creedmoor-100-gr-eld-vt-v-match` but the page content is actually 6.5 Creedmoor. This is a Hornady website routing issue — the data in our DB is correct. Low priority.

### Weight Mismatches (C1)
```sql
SELECT c.id, c.name, c.bullet_weight_grains, b.weight_grains, ABS(c.bullet_weight_grains - b.weight_grains)
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id
WHERE ABS(c.bullet_weight_grains - b.weight_grains) > 1.0;
-- Result: 72 rows (was 97)
```

### Diameter Mismatches
```sql
SELECT c.id, c.name, b.bullet_diameter_inches, cal.bullet_diameter_inches
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id JOIN caliber cal ON c.caliber_id = cal.id
WHERE ABS(b.bullet_diameter_inches - cal.bullet_diameter_inches) > 0.002;
-- Result: 0 rows
```

### BC Error Thresholds
```sql
SELECT b.id, b.name, m.name, b.bullet_diameter_inches, b.bc_g1_published, b.bc_g7_published
FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE (b.bc_g1_published IS NOT NULL AND (b.bc_g1_published < 0.05 OR b.bc_g1_published > 1.2))
   OR (b.bc_g7_published IS NOT NULL AND (b.bc_g7_published < 0.02 OR b.bc_g7_published > 0.6));
-- Result: 2 rows (Hornady .358 148gr HBWC, CE .510 1002gr MTAC — both legitimate)
```

### Missing BC Rifle Bullets (W2)
```sql
SELECT b.id, b.name, m.name, b.bullet_diameter_inches
FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE b.bc_g1_published IS NULL AND b.bc_g7_published IS NULL
  AND b.bc_g1_estimated IS NULL AND b.bc_g7_estimated IS NULL
  AND b.bullet_diameter_inches <= 0.375
  AND m.name NOT LIKE '%Cutting Edge%' AND m.name NOT LIKE '%Nosler%'
  AND b.id NOT IN (SELECT DISTINCT bullet_id FROM bullet_bc_source);
-- Result: 15 rows (Sierra 4, Lehigh 4, Federal 4, Lapua 3)
```
