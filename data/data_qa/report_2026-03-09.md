## Queries Run

### Weight Mismatches (97 results)
```sql
SELECT c.id, c.name, c.bullet_weight_grains AS cart_wt, b.weight_grains AS bullet_wt,
       ABS(c.bullet_weight_grains - b.weight_grains) AS diff
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id
WHERE ABS(c.bullet_weight_grains - b.weight_grains) > 1.0
ORDER BY diff DESC;
```

### Zero/Implausible Muzzle Velocity (13 results: 12 zero + 1 high)
```sql
SELECT c.id, c.name, c.muzzle_velocity_fps, m.name AS mfr, cal.name AS caliber
FROM cartridge c JOIN manufacturer m ON c.manufacturer_id = m.id
JOIN caliber cal ON c.caliber_id = cal.id
WHERE c.muzzle_velocity_fps <= 0 OR c.muzzle_velocity_fps > 4200
ORDER BY c.muzzle_velocity_fps;
```

### Missing BC Rifle Bullets (6 results, non-CE/Nosler, ≤.375" diameter)
```sql
SELECT b.id, b.name, m.name AS mfr, b.bullet_diameter_inches AS diam
FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE b.bc_g1_published IS NULL AND b.bc_g7_published IS NULL
  AND b.bc_g1_estimated IS NULL AND b.bc_g7_estimated IS NULL
  AND b.bullet_diameter_inches <= 0.375
  AND m.name NOT IN ('Cutting Edge Bullets', 'Nosler')
ORDER BY m.name, b.bullet_diameter_inches;
```

### Sierra TMK Bad Record Check
```sql
SELECT b.id, b.name, b.bullet_diameter_inches, b.weight_grains, b.bc_g1_published
FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE b.name LIKE '%TMK%' AND b.weight_grains = 60 AND m.name LIKE '%Sierra%';
-- Result: 2 records. Bad (diam=0.220, no BC) + Good (diam=0.224, G1=0.323)
```
