### C5: Hornady .300 Win Mag Custom International metric conversion error
- **Cartridge**: `300 Win Mag 180 GR SP Custom International™` (id: `59cf6217-b9fd-40a8-8745-da283a127b66`)
- **Barrel length**: Stored as **9.45"** — actually 24cm from Hornady International metric page, should be **24"** (standard .300 WM test barrel)
- **Muzzle velocity**: Stored as **2961 fps** — converted from 902 m/s. Correct conversion is 2959.3 fps; 2 fps rounding error
- **Source URL**: https://www.hornady.com/ammunition/rifle/300-win-mag-180-gr-sp-custom-international
- **Impact**: 9.45" barrel length is nonsensical for .300 WM and would produce wrong ballistic solutions if used
- **Same bug pattern** as resolved C3 (Hornady International metric confusion). Additional International cartridges should be audited.

## Queries Run

### Cross-Entity Weight Mismatches (68 rows)
```sql
SELECT c.id, c.name, c.bullet_weight_grains AS cart_wt, b.weight_grains AS bullet_wt,
       ABS(c.bullet_weight_grains - b.weight_grains) AS diff
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id
WHERE ABS(c.bullet_weight_grains - b.weight_grains) > 1.0
ORDER BY diff DESC;
```

### Bullet-Caliber Diameter Mismatch (0 rows)
```sql
SELECT c.id, c.name, b.bullet_diameter_inches AS bullet_diam,
       cal.bullet_diameter_inches AS cal_diam, cal.name AS caliber
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id JOIN caliber cal ON c.caliber_id = cal.id
WHERE ABS(b.bullet_diameter_inches - cal.bullet_diameter_inches) > 0.002;
```

### BC Error Thresholds (2 rows)
```sql
SELECT b.id, b.name, m.name AS mfr, b.bullet_diameter_inches AS diam,
       b.bc_g1_published AS g1, b.bc_g7_published AS g7
FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE (b.bc_g1_published IS NOT NULL AND (b.bc_g1_published < 0.05 OR b.bc_g1_published > 1.2))
   OR (b.bc_g7_published IS NOT NULL AND (b.bc_g7_published < 0.02 OR b.bc_g7_published > 0.6));
```

### Zero/Implausible Velocity (8 rows)
```sql
SELECT c.id, c.name, c.muzzle_velocity_fps, m.name AS mfr
FROM cartridge c JOIN manufacturer m ON c.manufacturer_id = m.id
WHERE c.muzzle_velocity_fps <= 0 OR c.muzzle_velocity_fps > 4200;
```

### Missing BCs - Rifle Bullets (22 rows)
```sql
SELECT m.name, COUNT(*) AS cnt
FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE b.bullet_diameter_inches <= 0.375
  AND b.bc_g1_published IS NULL AND b.bc_g7_published IS NULL
  AND b.bc_g1_estimated IS NULL AND b.bc_g7_estimated IS NULL
  AND m.name NOT LIKE '%Cutting Edge%' AND m.name NOT LIKE '%Nosler%'
GROUP BY m.name ORDER BY cnt DESC;
```
