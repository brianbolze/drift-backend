## New Critical Issues

### C9: Lapua G580 100gr bullet stored with wrong diameter
- **Bullet**: Lapua 6,5 g / 100 gr Open Tip G580 (id: `8e35868b`)
- **Issue**: DB has `bullet_diameter_inches = 0.264` (6.5mm), but manufacturer page confirms this is a **.30 cal (0.308")** bullet. The "6,5 g" in the name is the **weight in grams**, not the caliber.
- **Impact**: CRITICAL — bullet would be matched to 6.5mm cartridges instead of .308 cartridges, producing completely wrong ballistic solutions.
- **BC**: G1=0.200 is correct. Weight 100gr is correct.
- **Fix**: Update `bullet_diameter_inches` from 0.264 to 0.308 via curation patch.

## New Warnings

### W11: 90 newly ingested cartridges with zero muzzle velocity
- **Breakdown**: Winchester (43), Barnes (33), Nosler (13)
- **Root cause**: Barnes and Winchester do not publish velocity on their product pages. Nosler shop.nosler.com pages redirect. Pipeline extracted all other fields but couldn't get MV.
- **Impact**: These cartridges are unusable for ballistic calculations without velocity.
- **Recommendation**: Populate MV via curation patches using third-party retailer data (e.g., Barnes 308 110gr RR = 2910 fps, Barnes 6.5 PRC 130gr TSX = 3010 fps, Winchester 270 BST 130gr = 3050 fps, Winchester EBG 6.5 CM 142gr = 2700 fps — all confirmed from multiple retailers).

```sql
-- Referential integrity (all zero)
SELECT 'bullet_orphan_mfr', COUNT(*) FROM bullet b LEFT JOIN manufacturer m ON b.manufacturer_id = m.id WHERE m.id IS NULL
-- ... (6 checks, all zero)

-- Weight mismatches (11 found)
SELECT c.id, c.name, c.bullet_weight_grains, b.weight_grains, ABS(c.bullet_weight_grains - b.weight_grains) AS diff
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id
WHERE ABS(c.bullet_weight_grains - b.weight_grains) > 1.0;

-- Diameter mismatches (0 found)
SELECT c.id, c.name, b.bullet_diameter_inches, cal.bullet_diameter_inches
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id JOIN caliber cal ON c.caliber_id = cal.id
WHERE ABS(b.bullet_diameter_inches - cal.bullet_diameter_inches) > 0.002;

-- Zero MV (104 found)
SELECT c.id, c.name, c.muzzle_velocity_fps, m.name FROM cartridge c JOIN manufacturer m ON c.manufacturer_id = m.id
WHERE c.muzzle_velocity_fps = 0;

-- Missing BC rifle bullets (33 found)
SELECT m.name, COUNT(*) FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE b.bc_g1_published IS NULL AND b.bc_g7_published IS NULL AND b.bc_g1_estimated IS NULL AND b.bc_g7_estimated IS NULL
AND m.name NOT LIKE '%Cutting Edge%' AND m.name NOT LIKE '%Nosler%' AND b.bullet_diameter_inches <= 0.375
GROUP BY m.name;

-- BC source divergence (0 single-source issues)
SELECT b.name, bcs.bc_type, bcs.bc_value, ... FROM bullet b JOIN bullet_bc_source bcs ON bcs.bullet_id = b.id
WHERE (SELECT COUNT(*) FROM bullet_bc_source s2 WHERE s2.bullet_id = b.id AND s2.bc_type = bcs.bc_type) = 1
AND ABS(bcs.bc_value - published) > 0.001;
```
