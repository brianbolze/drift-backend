## Queries Run

### Referential Integrity
```sql
SELECT 'bullet->mfr' AS chk, COUNT(*) FROM bullet b LEFT JOIN manufacturer m ON b.manufacturer_id = m.id WHERE m.id IS NULL
UNION ALL SELECT 'cart->bullet', COUNT(*) FROM cartridge c LEFT JOIN bullet b ON c.bullet_id = b.id WHERE b.id IS NULL
UNION ALL SELECT 'cart->caliber', COUNT(*) FROM cartridge c LEFT JOIN caliber cal ON c.caliber_id = cal.id WHERE cal.id IS NULL
UNION ALL SELECT 'cart->mfr', COUNT(*) FROM cartridge c LEFT JOIN manufacturer m ON c.manufacturer_id = m.id WHERE m.id IS NULL
UNION ALL SELECT 'bcs->bullet', COUNT(*) FROM bullet_bc_source bcs LEFT JOIN bullet b ON bcs.bullet_id = b.id WHERE b.id IS NULL;
-- Result: all 0
```

### Weight Mismatches
```sql
SELECT c.id, c.name, c.bullet_weight_grains AS cart_wt, b.weight_grains AS bullet_wt,
       ABS(c.bullet_weight_grains - b.weight_grains) AS diff
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id
WHERE ABS(c.bullet_weight_grains - b.weight_grains) > 1.0;
-- Result: 8 rows (5 Federal Custom 0→various, .22 WMR 30→35, .308 155→160, 30-06 150→147)
```

### Diameter Mismatches
```sql
SELECT c.id, c.name, b.bullet_diameter_inches AS bullet_diam,
       cal.bullet_diameter_inches AS cal_diam, cal.name AS caliber
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id JOIN caliber cal ON c.caliber_id = cal.id
WHERE ABS(b.bullet_diameter_inches - cal.bullet_diameter_inches) > 0.002;
-- Result: 0 rows
```

### Zero/High MV
```sql
SELECT c.id, c.name, c.muzzle_velocity_fps, m.name AS mfr
FROM cartridge c JOIN manufacturer m ON c.manufacturer_id = m.id
WHERE c.muzzle_velocity_fps <= 0 OR c.muzzle_velocity_fps > 4200;
-- Result: 14 zero-MV + 2 high-MV (.204 Ruger)
```

### Missing BC Rifle Bullets
```sql
SELECT m.name, COUNT(*) FROM bullet b JOIN manufacturer m ON b.manufacturer_id = m.id
WHERE b.bc_g1_published IS NULL AND b.bc_g7_published IS NULL
  AND b.bc_g1_estimated IS NULL AND b.bc_g7_estimated IS NULL
  AND b.bullet_diameter_inches <= 0.375
  AND m.name NOT IN ('Cutting Edge Bullets', 'Nosler')
  AND b.id NOT IN (SELECT DISTINCT bullet_id FROM bullet_bc_source)
GROUP BY m.name ORDER BY COUNT(*) DESC;
-- Result: Winchester 15, Sierra 5, Federal 4, Lehigh 4, Lapua 3, Swift 1, Norma 1 = 33
```
