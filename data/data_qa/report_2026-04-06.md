## New Critical Issues

### C13: Sako TRG Precision .308 wrong weight and BC
- **Record**: Sako TRG Precision .308 174gr (id=afee55ff)
- **Issue**: Weight stored as 174gr, should be **175gr**. G1 BC stored as 0.472, should be **0.467**.
- **Source**: Curated data from gap-fill session. Multiple retailer sources (TargetSportsUSA, others) consistently show 175gr Scenar-L HPBT with G1=0.467.
- **Impact**: Both weight and BC are wrong, affecting ballistic calculations for anyone using this bullet.
- **Fix**: Curation patch needed. Related Sako entries (Powerhead Blade .308 174gr, Powerhead Blade Pro .308 174gr) may also have incorrect weight — Sako's SPA-only website prevents direct verification.
- **Added to**: `known_issues.json` (C13), `TODO.md`

-- Referential integrity (combined, all 6 checks)
SELECT 'orphan_bullet_mfr' AS check_type, COUNT(*) FROM bullet b LEFT JOIN manufacturer m ON b.manufacturer_id = m.id WHERE m.id IS NULL
UNION ALL SELECT 'orphan_cart_bullet', COUNT(*) FROM cartridge c LEFT JOIN bullet b ON c.bullet_id = b.id WHERE b.id IS NULL
UNION ALL SELECT 'orphan_cart_caliber', COUNT(*) FROM cartridge c LEFT JOIN caliber cal ON c.caliber_id = cal.id WHERE cal.id IS NULL
UNION ALL SELECT 'orphan_cart_mfr', COUNT(*) FROM cartridge c LEFT JOIN manufacturer m ON c.manufacturer_id = m.id WHERE m.id IS NULL
UNION ALL SELECT 'orphan_bcs_bullet', COUNT(*) FROM bullet_bc_source bcs LEFT JOIN bullet b ON bcs.bullet_id = b.id WHERE b.id IS NULL
UNION ALL SELECT 'orphan_alias', COUNT(*) FROM entity_alias ea WHERE (ea.entity_type = 'manufacturer' AND ea.entity_id NOT IN (SELECT id FROM manufacturer)) OR (ea.entity_type = 'caliber' AND ea.entity_id NOT IN (SELECT id FROM caliber)) OR (ea.entity_type = 'bullet' AND ea.entity_id NOT IN (SELECT id FROM bullet));
-- Result: All 0

-- Weight mismatches
SELECT c.id, c.name, c.bullet_weight_grains, b.weight_grains, ABS(c.bullet_weight_grains - b.weight_grains) AS diff
FROM cartridge c JOIN bullet b ON c.bullet_id = b.id WHERE ABS(c.bullet_weight_grains - b.weight_grains) > 1.0;
-- Result: 0 rows
